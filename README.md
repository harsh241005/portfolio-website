# Harsh Palyekar - Data Science Portfolio 🚀

Welcome to my personal portfolio website! This project showcases my skills in Data Science, Machine Learning, and Business Analytics, along with my featured projects, technical expertise, and contact information.

## 🌟 Features
- **Modern & Premium Design**: Built with a sleek dark-mode glassmorphism theme and stylized typography.
- **Interactive 3D Background**: Features a custom-built, interactive 3D particle network representing neural connections and data clouds, powered by `Three.js`.
- **Responsive Layout**: Designed to look great on desktops, tablets, and mobile devices.
- **Project Showcase**: Highlights my end-to-end Machine Learning systems, data analysis, and predictive modeling projects.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Vanilla, Glassmorphism design)
- JavaScript (ES6+)
- [Three.js](https://threejs.org/) (for the interactive 3D background animation)

## 💻 Local Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/harsh241005/portfolio-website.git
   ```
2. Open the folder and run a local server. For example, using Python:
   ```bash
   python -m http.server 8080
   ```
3. Open `http://localhost:8080` in your web browser.
