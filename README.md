## 🚀 Featured Projects

### 📊 InsightX – E-Commerce Analytics Platform
**Tech Stack:** Python, SQL, BigQuery, BigQuery ML, Streamlit, Google Cloud Platform

- Built an end-to-end analytics pipeline processing **100K+** e-commerce transactions.
- Developed customer segmentation using **K-Means**.
- Built sales forecasting models using **BigQuery ML**.
- Designed an interactive multi-page Streamlit dashboard.
- Deployed on Google Cloud using Docker and Cloud Run.

---

### 📈 Socio-Technical Health Monitor
**Tech Stack:** Python, NLP, XGBoost, Streamlit

- Built a seven-stage NLP pipeline for project health prediction.
- Processed over **112,000** email-ticket relationships.
- Compared Logistic Regression, Random Forest, and XGBoost.
- Implemented SHAP explainability for model interpretation.

---

### 🏏 IPL Analytics Dashboard
**Tech Stack:** Power BI, SQL, Excel

- Interactive dashboard for IPL performance analysis.
- Team, player, and season-level insights.
- KPI cards and dynamic filters.
- Business-focused visualizations.

---

### 📉 Telecom Churn Profit Evaluation
**Tech Stack:** Python, Scikit-learn

- Compared multiple machine learning algorithms.
- Evaluated models based on **business profit**, not just accuracy.
- Demonstrated business-driven model selection.

---

### 🚗 Indian Automobile Enterprise Database
**Tech Stack:** Oracle SQL, Python

- Designed a normalized Oracle SQL database.
- Implemented advanced SQL using Window Functions, ROLLUP, DENSE_RANK, and OLAP queries.
- Built Python visualizations for sales analysis.
